# NWSS COE Resource Repository

A centralized directory of resources developed by Centers of Excellence (COEs) participating in the CDC National Wastewater Surveillance System (NWSS). This repository is maintained by the **New York NWSS COE** on behalf of the broader COE network.

Its purpose is to reduce duplication of effort across the network by making existing tools, materials, and technical resources discoverable and, where permitted, adaptable by other COEs, health departments, utilities, and federal partners.

---

## How to use this directory

Browse by COE using the directory below, or use GitHub's search to find resources by keyword — for example, `risk communication`, `utility retention`, or `influenza`. Each COE entry lists resource categories, access requirements, formats, and direct links.

**To submit your COE's resources:** See [CONTRIBUTING.md](CONTRIBUTING.md).

**To report a broken link or request a correction:** Open an issue tagged `correction`.

**Questions:** Open an issue tagged `question` or contact the repository maintainer directly.

---

## Resource categories

This directory uses the following standardized categories. COEs may offer resources in one or more areas.

| Category | Description |
|----------|-------------|
| Data coding and analysis | Code, scripts, and analytical frameworks for processing wastewater surveillance data |
| Risk communication | Materials for communicating surveillance findings to public, media, or decision-makers |
| Utility retention and engagement | Tools and strategies for maintaining utility participation in NWSS |
| How-to guides and SOPs | Step-by-step guidance documents and standard operating procedures |
| Peer-to-peer technical assistance | Direct TA offerings available to other COEs or jurisdictions |
| Use cases and applications | Examples of how wastewater data has been applied to public health action |
| Success stories | Documented outcomes and program highlights from cohort jurisdictions |
| Trainings and workshops | Recorded or live training offerings, curricula, and workshop materials |
| Data visualization and dashboards | Interactive tools and visual displays of surveillance data |
| Funding and resource navigation | Guidance on funding mechanisms, grant resources, and programmatic sustainability |
| Lab methods and protocols | Laboratory procedures, quality assurance documents, and method validation resources |
| Equity and access resources | Tools addressing equitable surveillance coverage and inclusive program design |
| Surveillance reports and briefs | Periodic reports, situation reports, and technical summaries |
| Policy and translation tools | Resources bridging surveillance data and policy or programmatic decision-making |
| Presentations and webinars | Slide decks, recorded presentations, and webinar archives |
| Publications and manuscripts | Peer-reviewed articles, preprints, and technical reports |
